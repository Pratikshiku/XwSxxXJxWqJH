### 三、功能描述
系统分为（身份）：普通用户（住户）、维修人员、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 房间信息
6. 维修申请
7. 维修记录
8. 维修评价


**维修人员功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 维修分配
5. 改派申请
6. 请假信息
7. 维修记录
8. 维修评价



**管理员功能：**
1. 登录
2. 个人中心
3. 住户管理
4. 房间信息管理
5. 维修人员管理
6. 维修分类管理
7. 物品信息管理
8. 维修申请管理
9. 维修分配管理
10. 改派申请管理
11. 请假信息管理
12. 维修记录管理
13. 维修评价管理
